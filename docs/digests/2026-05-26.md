## Top Stories

### [California moves to exempt Linux from its age-verification law after backlash](https://news.ycombinator.com/item?id=48269961)
California's controversial age-verification law (AB 1043) is facing an amendment to exempt open-source operating systems from collecting user ages. The HN discussion highlights the absurdity of the mandate and suspects the exemption is a tactical legal move to prevent Linux developers from challenging the law on First Amendment grounds.

### [Using AI to write better code more slowly](https://news.ycombinator.com/item?id=48272984)
A pragmatic pushback against the "10x productivity" hype of AI coding. The author argues for using LLMs as tireless reviewers and critics rather than autonomous code generators. The comments reveal that many senior engineers have settled on this exact workflow: using AI for deep code review and finding corner cases, which results in better code but takes just as much time.

### [CVE-2026-28952: Apple macOS 26.5 Kernel Vuln found by Claude](https://news.ycombinator.com/item?id=48273169)
Anthropic's Claude (specifically the Mythos model) discovered a zero-day integer overflow in the macOS kernel. The thread reflects on how AI is accelerating vulnerability discovery, shifting the engineering culture from treating server uptime as a badge of honor to viewing anything unpatched for a week as a massive security risk.

### [Exit IP VPN servers mitigation rollout](https://news.ycombinator.com/item?id=48269580)
Mullvad quickly rolled out a mitigation for a fingerprinting vulnerability between VPN exit servers. The community praised the company's prompt and transparent technical explanation, noting how refreshing it is to see a small company act decisively compared to the bureaucratic inertia of larger tech behemoths.
