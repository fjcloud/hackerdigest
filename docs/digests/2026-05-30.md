## Top Stories

### [The dead economy theory](https://news.ycombinator.com/item?id=48324712)
An essay arguing that AI automation creates a prisoner's dilemma: firms rationally cut labor costs, but collectively destroy the consumer base needed to buy their products. The HN discussion is a sharp debate on whether this wave of automation is different. The best counter-argument notes that while past automation replaced physical labor, AI aims to automate intelligence itself, potentially leaving no safe harbor for displaced workers.

### [SQLite is all you need for durable workflows](https://news.ycombinator.com/item?id=48326802)
A proposal to use SQLite for agent orchestration and state management instead of complex distributed systems like Temporal. The community is split. Proponents praise the simplicity of treating workflows as local state, while skeptics point out that once you add Litestream and object storage for actual durability, you have reinvented a distributed system anyway.

### [Is AI causing a repeat of frontend’s lost decade?](https://news.ycombinator.com/item?id=48321631)
The author argues that AI code generation is deskilling frontend development, much like heavy frameworks did previously. HN commenters strongly push back, noting that the "golden age" of frontend was mostly wrestling with browser quirks and accidental complexity. Many argue that automating repetitive UI tasks is a net positive that frees engineers to focus on actual user experience.
