## Top Stories

### [Anthropic penalizes Claude Code users for mentioning competitor](https://news.ycombinator.com/item?id=47963204)
Anthropic's Claude Code CLI is abruptly terminating sessions or charging extra usage if a user's git commits contain the string "OpenClaw" (a competing open-source harness). The heavy-handed, regex-based blocking has sparked outrage, with commenters noting it feels like a desperate move from a compute-constrained company damaging its own reputation.

### [PyTorch Lightning compromised in supply chain attack](https://news.ycombinator.com/item?id=47964617)
The PyPI credentials for the popular PyTorch Lightning library were leaked, allowing attackers to publish malicious versions (2.6.2 and 2.6.3) containing credential-stealing malware. The incident highlights the ongoing fragility of the Python ecosystem, with commenters noting that AI coding assistants often blindly suggest installing these compromised packages.

### [Linux kernel vulnerability disclosed without distro heads-up](https://news.ycombinator.com/item?id=47965108)
A researcher publicly dropped a Linux kernel exploit before distributions like Ubuntu or RHEL had patches ready, seemingly to market their AI security product. The incident sparked a fierce debate on HN about disclosure norms, with some arguing end-users have a right to know immediately, while others condemned it as an irresponsible marketing stunt that put shared hosting providers at risk.
