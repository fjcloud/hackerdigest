## Top Stories

### [Zerostack – A Unix-inspired coding agent written in pure Rust](https://news.ycombinator.com/item?id=48164287)
A new open-source coding agent that boasts an 8MB RAM footprint, a stark contrast to the gigabytes consumed by tools like Claude Code. The HN thread highlights a growing fatigue with bloated, memory-leaking agent harnesses, with commenters noting that building custom, lightweight agent loops is becoming the new "writing your own web framework."

### [Moving away from Tailwind, and learning to structure my CSS](https://news.ycombinator.com/item?id=48158400)
Julia Evans wrote about her transition from Tailwind back to semantic HTML and structured CSS. The resulting HN thread is a massive, polarized debate on the merits of utility classes versus separation of concerns. An interesting sub-thread notes that frontier LLMs strongly prefer Tailwind because coupling style definitions directly to document structure requires less "cognitive" overhead than reasoning about cascading stylesheets.

### [Accelerando (2005)](https://news.ycombinator.com/item?id=48159241)
A retrospective on Charles Stross's 2005 sci-fi novel about the singularity. Commenters are struck by how prescient the book feels today, particularly its depiction of humans losing their agency to AI assistants, and corporations run entirely by AI agents suing each other in milliseconds to exhaust compute resources. As one reader noted, the "skills atrophy" described in the book is already happening.
