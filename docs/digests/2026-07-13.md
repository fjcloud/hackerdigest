## Top Stories

### [Claude Code’s 33k-token startup cost is real, but the benchmark is incomplete](https://news.ycombinator.com/item?id=48883275)
A benchmark measured a 33k-token initial payload for Claude Code versus 7k for OpenCode. It used an older model and a gateway that altered requests, so quality and cache-adjusted costs remain unproven. One commenter saw seven subagents exhaust a five-hour quota before any finished, while sequential execution succeeded.

### [GPT-5.6 migration exposed how provider-specific production agents remain](https://news.ycombinator.com/item?id=48882716)
Ploy moved its website-building agent from Opus to GPT-5.6, cutting wall time 2.2x and cost 27% while preserving its evaluations. The migration required explicit cache keys and nullable tool fields. That supports the thread’s warning that model, prompt, and harness form one coupled system.

### [Google Maps tested load-balancing traffic across near-equal routes](https://news.ycombinator.com/item?id=48881967)
Google Maps deliberately distributed drivers across near-equal routes in a six-month experiment spanning ten US cities. Targeted road speeds rose about 2%, while estimated fuel use fell 0.5–1%. Commenters flagged the externalities: residential spillover, accelerated road wear, and users abandoning an app that optimizes the network over their trip.
