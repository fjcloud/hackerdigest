## Top Stories

### [Chat Control 1.0 and 2.0 Explained](https://news.ycombinator.com/item?id=48818311)
EU Parliament triggered Chat Control 1.0's reactivation yesterday in Strasbourg, months after the ePrivacy derogation expired on April 4, 2026. The technical reality of "control 2.0" for end-to-end-encrypted messengers: client-side scanning on-device, with `arjie` summarizing the fork as "MITM by privileged authority" vs "Apple-style CSAM scanner." Signal has stated it will leave the EU market if 2.0 becomes law. The thread quickly turns into a referendum on the EU itself -- multiple commenters call it "the surveillance capital of the world," and Signal's leaving-vs-complying post is the most-circulated reference.

### [EU mandates driver monitoring cameras in every new car](https://news.ycombinator.com/item?id=48823557)
From July 2026, all new EU cars must ship with an Advanced Driver Distraction Warning system -- typically a face-tracking camera. The closed-loop spec means no biometric storage or transmission (per the reg text), but the practical effects thread is brutal: `Dries007` reports Volvo's system misidentifies "singing" as "yawning" and triggers cascading alerts. `dmitrygr` invokes the aviation "normalization of deviance" pattern. Several commenters note the eCall modem is already mandatory and illegal to disable -- the camera is a precedent, not an exception.

### [StreetComplete: fix OpenStreetMap, one tiny quest at a time](https://news.ycombinator.com/item?id=48816883)
The highest-voted story of the day at 730 points. StreetComplete hides OSM's tagging model behind a quest system: a beginner walks around with the app and answers "is this utility pole wood or steel?" The community has clearly internalized that hiding the schema is the UX move -- `wxlong2000` puts it well: "Most contributor tools expose the schema too early." Related in-thread: Every Door and SCEE for power users, SCEE for adding more aggressive tasks. The map-as-public-good argument is real: `GreyStache` mapped a new road and it appeared in Google Maps within a week.

### [30papers.com -- Ilya's 30 essential ML papers, in beginner-friendly format](https://news.ycombinator.com/item?id=48819608)
A first-year CS student at Trinity College Dublin reformatted the famous "Ilya gave Carmack a reading list" 30 papers into a single site, then argued with HN about it. The actual list provenance is murky -- the OP says it came from ex-OpenAI employee Andrew Carr, and the community flags the original X post by `@keshavchan` from 2024 that may have been community-collated. The site itself is widely panned for layout choices (severe motion sickness reported) and broken LaTeX rendering. A more useful artifact in-thread: `cute_boi`'s plain list of all 30 papers with arxiv links.

### [Why we built yet another Postgres connection pooler](https://news.ycombinator.com/item?id=48819308)
PgDog is a new Rust/AGPL Postgres pooler that goes beyond pgbouncer: it fixes `NOTIFY/LISTEN` across pool boundaries, auto-routes to read replicas, supports prepared statements, and (eventually) sharding. The interesting HN discussion is about connection state leakage: `petters` is shocked that `SET` commands and cursor state can persist across pooled clients. `danudey` shows the same problem on MySQL with a charset mismatch example, and ClickHouse's blog on scaling pgbouncer (linked in-thread) is recommended companion reading.

## Projects & Tools

- **[Davit: Apple Containers UI](https://news.ycombinator.com/item?id=48821848)** -- A 17MB native SwiftUI app over Apple's new OCI-compatible container runtime, with a menubar UI, signed/notarized, and 28 commits in 3 days (every commit co-authored by Fable). `simonw` notes the API is built on `ContainerAPIClient` directly with no Electron, and the platform services idle at 25MB vs Docker Desktop's permanent VM. Watch the legal notes on Fable co-authorship in the README.
- **[l: A new runtime for k and q](https://news.ycombinator.com/item?id=48821378)** -- The "next letter after k" in the K/Q ecosystem: a closed-source runtime for the language Wall Street uses for tick data, with full kdb+ compatibility claims and "compute on compressed data" as the architectural bet. The site is vibecoded and the community is appropriately skeptical, but the underlying design point -- memory bandwidth is the wall, so compress the array -- is real and worth understanding.
- **[Herdr: One terminal to rule them all](https://news.ycombinator.com/item?id=48756578)** -- Rust TUI multiplexer pitched as a tmux rewrite aimed at "I run 10 agents in parallel" workflows. Killer features per users: agent status indicators (idle/working) and automatic git worktree management. The "Popular with engineers from..." logo wall drew the usual HN scorn. Treat as one of several contenders (cmux, zmx, Conductor) in the agent-native terminal race.
- **[Jim's TrueType QR Code Font](https://news.ycombinator.com/item?id=48820119)** -- A font where the text you select is what gets copied, but the rendered glyphs form a valid QR code. OpenType GSUB does glyph stacking and Reed-Solomon error correction at rasterization time. `foodevl` (the author) admits the design was LLM-generated; the result is a real font, not a toy. The page warns about Safari/space bugs.

## Worth Reading

- **[AI Meets Cryptography 1: What AI Found in Cloudflare's Circl](https://news.ycombinator.com/item?id=48821749)** -- Trained an LLM audit pipeline (`zkao.io`) at Cloudflare's crypto library; 7 real bugs across threshold RSA, CP-ABE, and BLS. The headline bug: `float64` in RSA key generation with a 53-bit mantissa causing silent rounding. `mcpherrinm` notes Falcon (FN-DSA) is the one modern signature scheme where float is actually required, which makes the Circl bug worse -- the lesson is that "I'm using floats" needs to be obviously justified in crypto code.
- **[Local, CPU-Friendly, High-Quality TTS with Kokoro](https://news.ycombinator.com/item?id=48821576)** -- Kokoro at 82M params is now the default recommendation for local TTS. `mowmiatlas` got it running on iPhone ANE, `dvt` has it in a 85MB WASM build for browser games. Trade-off surfaced in-thread: short utterances get the wrong intonation (the fix is to ask for a longer sentence, then crop using the word timestamps).
- **[Tenda firmware backdoor (CVE)](https://news.ycombinator.com/item?id=48825749)** -- Hardcoded `sys.rzadmin.username=rzadmin` / password `rzadmin` (base64-obfuscated) across multiple router firmware versions. `dhx` does the actual binwalk in-thread and finds the same pattern on a model the CERT advisory didn't list. CNIL-style mandatory reporting isn't on the table; commenters note the asymmetric enforcement ("find and publish customer emails = FBI raid, ship a router backdoor = nothing").
- **[How to Build a Minimal ZFS NAS Without Synology, QNAP, TrueNAS](https://news.ycombinator.com/item?id=48827325)** -- Walkthrough of a Debian + ZFS + Samba stack on a Jonsbo N6. Good context: `Confiks` is doing the same build and notes TFA's ECC recommendation is largely cargo-cult, the `dnodesize=auto` default has known footguns, and shucked 14TB WD Elements are the best $/TB available right now. The FreeBSD-vs-Linux ZFS religious war resurfaces intact.
