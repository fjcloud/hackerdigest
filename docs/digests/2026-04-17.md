## Top Stories

### [Claude Opus 4.7](https://news.ycombinator.com/item?id=47793411)
Anthropic released Opus 4.7 with a new tokenizer and higher default reasoning effort. The HN thread is dominated by frustration over aggressive new cybersecurity filters that block legitimate reverse-engineering workflows, and complaints that the new tokenizer burns through usage limits significantly faster.

### [Codex for almost everything](https://news.ycombinator.com/item?id=47796469)
OpenAI launched a major update to the Codex desktop app, adding "computer use" capabilities that allow the agent to click and type across macOS. While some welcome the automation, the discussion focuses heavily on the security implications of giving an LLM full desktop control without a robust sandbox.

### [Qwen3.6-35B-A3B: Agentic coding power, now open to all](https://news.ycombinator.com/item?id=47792764)
Alibaba released the open-weights Qwen 3.6 MoE model (35B total, 3B active parameters). Engineers note it is a capable local model for agentic coding that runs comfortably on consumer hardware, offering a private, uncapped alternative to rate-limited commercial APIs.

### [New unsealed records reveal Amazon's price-fixing tactics](https://news.ycombinator.com/item?id=47800166)
Unsealed records from the California AG's lawsuit detail how Amazon punishes third-party sellers who offer lower prices on competing sites. Sellers in the comments confirm that Amazon's algorithm suppresses listings if a cheaper price is found elsewhere, effectively forcing merchants to raise prices across the entire internet to account for Amazon's high fees.
