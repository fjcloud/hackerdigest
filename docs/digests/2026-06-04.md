## Top Stories

### [Gemma 4 12B: A unified, encoder-free multimodal model](https://news.ycombinator.com/item?id=48385906)
Google released a new 12B parameter model that drops the traditional vision/audio encoders, feeding raw inputs directly into the LLM backbone. It's small enough to run locally on 16GB of RAM. The HN thread notes this architectural shift is a huge win for edge compute and single-board computers, though early testers report its vision capabilities still lag behind competitors like Qwen.

### [Elixir v1.20: Now a gradually typed language](https://news.ycombinator.com/item?id=48388324)
Elixir has officially introduced gradual typing. Rather than bolting on new syntax like TypeScript, the compiler infers precise type information from existing language constructs like pattern matching and guards. Commenters are praising the mature rollout, noting it catches bugs at compile time while emitting bytecode that is semantically identical to untyped code.

### [Uber's $1,500/month AI limit is a useful signal for AI tool pricing](https://news.ycombinator.com/item?id=48383056)
Uber has capped its engineers' AI API spending at $1,500 per month. The discussion highlights the stark reality of enterprise API pricing versus subsidized $20/month consumer plans. While some see the cap as a sign of diminishing returns on AI coding tools, others argue it's simply a necessary guardrail against developers running unoptimized, token-hungry autonomous agents overnight.

### [Failing grades soar with AI usage, dwindling math skills in Berkeley CS classes](https://news.ycombinator.com/item?id=48392004)
Introductory CS classes at UC Berkeley are seeing failure rates spike from under 10% to over 35%. Professors blame a combination of poor math fundamentals and students using LLMs to breeze through homework, leaving them entirely unprepared for exams. The comments reflect a growing concern about "cognitive atrophy" among developers who over-leverage AI instead of struggling through hard problems.
