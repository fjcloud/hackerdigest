## Top Stories

### [Apple's SpeechAnalyzer API outbenchmarks Whisper on-device](https://news.ycombinator.com/item?id=48894752)
A production app benchmarked Apple's new SpeechAnalyzer against the older SFSpeechRecognizer and Whisper on identical LibriSpeech audio. WER drops 3.5-4x on clean speech (2.12% vs 9.02%) at roughly 3x Whisper Small's speed. Commenters flagged the missing comparisons to Parakeet TDT, Whisper Large v3 Turbo, and Voxtral, and the run was English-only on one machine.

### [Building and shipping Apple apps without ever opening Xcode](https://news.ycombinator.com/item?id=48896665)
Walkthrough of xcodebuild plus notarytool plus fastlane pipelines that Claude Code drives end-to-end. A seven-year Xcode team vet confirmed it matches their own workflow. The strongest counter: bundle IDs and provisioning profiles still demand the GUI, no matter how good your scripts are.

### [Microsoft Claude Code / Copilot study: 24% more merged PRs, but at what cost?](https://news.ycombinator.com/item?id=48899321)
ArXiv preprint finds Microsoft adopters merged 24% more PRs over four months, and one heavy user burned through 281B tokens (around $1.4M at list). The thread's consensus is that throughput is the wrong metric: reviewers now spend more time clearing AI-generated queues than writing code.
