## Top Stories

### [LG monitors silently install software through Windows Update](https://news.ycombinator.com/item?id=48956688)
LG monitors install a McAfee-promoting package when connected to a Windows PC over HDMI, with no user prompt or consent. The package ships system-wide via Windows driver metadata and starts on every boot. Commenters noted Microsoft is the actual enabler. The same auto-install path is used by Razer, Logitech, and others.

### [GPT-5.6 used a prompt to close a 30-year gap in convex optimization](https://news.ycombinator.com/item?id=48957779)
A mathematician used a ten-page prompt and GPT-5.6 Sol Pro to prove a quadratic lower bound in convex Lipschitz optimization, then verified it in Lean. The thread pushed back on the "148 minutes" framing. The author had worked the problem for a year with prior models, and the prompt seeded the technique that worked.

### [Fable 5 vs. GPT-5.6 Sol on an NP-Hard problem](https://news.ycombinator.com/item?id=48956879)
A practitioner eval of the `/goal` directive on a TSP-like benchmark finds it gives both models a small or statistically insignificant boost. The deeper thread takeaway: agent quality collapses well before context maxes out. One engineer noted Codex's 400k cap is the right neighborhood, with most of the context still reliable.
