## Top Stories

### [Postmortem: TanStack NPM supply-chain compromise](https://news.ycombinator.com/item?id=48100706)
A sophisticated NPM worm hijacked CI/CD pipelines to compromise TanStack and other packages. The attackers poisoned GitHub Actions caches via the dangerous `pull_request_target` trigger, stealing developer secrets and installing a vindictive dead-man's switch that wipes the victim's machine if the stolen token is revoked. The HN thread highlights a critical flaw in modern CI: Trusted Publishing is insufficient if the pipeline itself can be compromised to push malicious releases.

### [GitLab announces workforce reduction and end of their CREDIT values](https://news.ycombinator.com/item?id=48100500)
GitLab is laying off staff, flattening management, and dropping its historical "CREDIT" values (which included Transparency and DEI) to pivot toward an "agentic era" of AI-driven development. The announcement claims they are reengineering Git for "machine scale." Commenters are highly skeptical, noting that GitLab is alienating its core enterprise user base by chasing AI hype rather than capitalizing on GitHub's recent stability issues.

### [Google says criminal hackers used AI to find a major software flaw](https://news.ycombinator.com/item?id=48094641)
Google's Threat Intelligence Group reported that attackers likely used an LLM to discover and weaponize a zero-day vulnerability in a popular open-source tool. This marks a shift in the offensive cybersecurity landscape. How did Google know it was AI? The comments point to the telltale signs left in the exploit script: hallucinated CVSS scores, excessive "educational" docstrings, and textbook Python formatting typical of LLM training data.
