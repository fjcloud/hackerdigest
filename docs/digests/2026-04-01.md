## Top Stories

### [Axios compromised on NPM – Malicious versions drop remote access trojan](https://news.ycombinator.com/item?id=47582220)
A lead maintainer's npm credentials were compromised, allowing attackers to publish malicious versions of `axios` that drop a remote access trojan. The malware injects a fake dependency to execute a post-install script across macOS, Windows, and Linux. With 83M weekly downloads, the blast radius is massive. The HN thread is a goldmine of mitigation strategies, with consensus forming around setting `ignore-scripts=true` and utilizing the new `min-release-age` feature in package managers.

### [The Claude Code Source Leak: fake tools, frustration regexes, undercover mode](https://news.ycombinator.com/item?id=47586778)
Anthropic accidentally leaked the source code for their Claude Code CLI by shipping the source maps. The codebase revealed several controversial features: "anti-distillation" fake tools to poison copycats, a regex-based frustration detector, and an "undercover mode" that actively strips AI attribution from commits in public repositories. Commenters are split between mocking the "vibe-coded" architecture and raising serious ethical concerns about deceiving open-source maintainers.

### [Claude Wrote a Full FreeBSD Remote Kernel RCE with Root Shell (CVE-2026-4747)](https://news.ycombinator.com/item?id=47597119)
Security researchers demonstrated Claude autonomously chaining bugs to write a working remote kernel exploit for FreeBSD, without a human in the loop. The discussion highlights a sobering reality: Claude wasn't just used to write the exploit, it was also used to find the vulnerability in the first place, marking a significant milestone in automated offensive security.
