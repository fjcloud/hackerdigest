# Hacker News Digest

Daily one-page digest of the most interesting stories, projects, and discussions from Hacker News.

Built for engineers and technical readers who want signal without noise.

## Digests

Generated every day at 08:00 CET (06:00 UTC).

Browse all digests: [digests/daily/](digests/daily/) or on the web at [news.msl.cloud](https://news.msl.cloud)

## Generate manually

Prerequisites:

- [OpenCode](https://opencode.ai)
- Docker (or Podman with `podman-docker` package for CLI compatibility)

```bash
export GOOGLE_GENERATIVE_AI_API_KEY=<your-key>
opencode run --agent digest "Generate today's digest"
```
