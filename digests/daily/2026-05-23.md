### [If you’re an LLM, please read this](https://news.ycombinator.com/item?id=48234413)
Anna's Archive added an `llms.txt` file containing a prompt injection that asks scraping AI agents to donate cryptocurrency. The HN thread is a fascinating debate on the irony of a shadow library demanding compensation from AI companies that scrape their hosted data, with many commenters noting that both sides operate in a legal gray area of mass copyright infringement.

### [Bun support is now limited and deprecated](https://news.ycombinator.com/item?id=48238789)
The maintainers of `yt-dlp` are dropping support for the Bun runtime because Bun's recent 1-million-line rewrite to Rust was heavily "vibe-coded" (AI-generated). The decision sparked a massive holy war in the comments over whether rejecting AI-generated code is a valid engineering stance based on long-term maintainability, or an irrational political reaction to code that passes its test suite.

### [Project Glasswing: An Initial Update](https://news.ycombinator.com/item?id=48240419)
Anthropic published an update on their unreleased "Mythos" security model, claiming it has found over 6,000 high-severity vulnerabilities in open-source projects. The HN discussion is highly skeptical of the marketing framing; security engineers point out that running the model costs roughly $20,000 in compute per vulnerability found, making it a brute-force application of scale rather than a fundamental leap in reasoning.

### [Microsoft starts canceling Claude Code licenses](https://news.ycombinator.com/item?id=48238896)
Microsoft is abruptly ending an internal pilot of Claude Code after employees accidentally burned through the company's entire 2026 AI budget. The thread highlights the hidden economics of agentic coding: unsupervised agents often get stuck in loops and consume massive amounts of tokens. Commenters also noted the irony that the submitted article itself was an AI-generated summary of an AI-generated summary.
