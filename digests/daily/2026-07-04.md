### [Leanstral 1.5: Proof abundance for all](https://news.ycombinator.com/item?id=48780801)
Mistral released Leanstral 1.5, a model specialized for generating formal proofs in the Lean theorem prover, alongside a claimed real-world catch: an integer overflow at U64_MAX in a Rust crate. Skeptics arrived quickly, noting that GPT-5.5 independently spotted the same bug when given the same repo, suggesting the find reflects sparse code review more than exceptional model capability.
