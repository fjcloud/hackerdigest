## Top Stories

### [Cursor 0day: full disclosure after six months of silence](https://news.ycombinator.com/item?id=48910676)
Mindgard published after six months and 197 Cursor releases with no fix. Cursor on Windows resolves `git.exe` from the workspace before PATH, so any cloned repo with a planted `git.exe` runs attacker code on open, no prompt. Commenters note VS Code uses fully qualified paths. Cursor's CISO was tagged on LinkedIn with no response.

### [Bonsai 27B: a 1.58-bit model that fits on a phone](https://news.ycombinator.com/item?id=48910545)
Prism ML's ternary weights compress a Qwen 27B to about 3.8 GB while keeping roughly 90% of the original. Runs in oMLX on M1 Pro. CNBC reports Apple is in talks with Prism. One independent benchmarker found the 4-bit quant collapses on GSM8K, so the on-device story leans on the ternary path.

### [Tailscale SSH permitted root login via leading-dash usernames](https://news.ycombinator.com/item?id=48915004)
TS-2026-009: logging in as `-i` yielded root. Tailscale fixed by rejecting dashed usernames rather than using `--`. The classic 1980s AIX bug class, still alive in 2026 Go rewrites. Found by Ada Logics using Anthropic's Mythos.

### [The Tower Keeps Rising](https://news.ycombinator.com/item?id=48909785)
An essay on lucumr.pocoo.org argues AI lets code keep growing past the point where humans share a mental model. Unlike Babel, the tower does not fall because agents fill the gaps. Top reply: agent work holds up in domains you understand, and produces tech debt at speed on unfamiliar stacks.
