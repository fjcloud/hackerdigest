## Top Stories

### [Anthropic no longer allowing Claude Code subscriptions to use OpenClaw](https://news.ycombinator.com/item?id=47633396)
Anthropic is cracking down on users plugging their $200/mo Claude Code subscriptions into third-party agent harnesses like OpenClaw, requiring them to use pay-as-you-go API billing instead. The HN thread highlights the tension between flat-rate subscription economics and power users running autonomous agents, with many developers threatening to switch to open models or OpenAI to maintain control over their workflows.

### [Delve removed from Y Combinator](https://news.ycombinator.com/item?id=47634690)
Y Combinator has scrubbed the compliance startup Delve from its website following allegations that the company was faking SOC2 audits and violating open-source licenses. The community notes that while "fake it till you make it" is a Silicon Valley staple, faking regulatory compliance crosses the line into outright fraud—especially when it exposes other YC companies to liability.

### [We replaced RAG with a virtual filesystem for our AI documentation assistant](https://news.ycombinator.com/item?id=47618223)
Mintlify abandoned vector-based RAG in favor of a virtual filesystem that allows their AI agent to navigate documentation using standard POSIX tools like `ls` and `grep`. The key insight from the discussion: LLMs are heavily trained on shell interactions, making a simple directory hierarchy a far more effective knowledge graph than complex embedding retrieval.

### [OpenClaw privilege escalation vulnerability](https://news.ycombinator.com/item?id=47628608)
A severe vulnerability in the popular OpenClaw AI agent allowed any user with basic access to escalate to admin privileges due to a missing authorization check. The thread serves as a stark warning about the current state of AI tooling: shipping at the speed of inference is leading to massive security oversights in tools that are given deep access to user systems.
