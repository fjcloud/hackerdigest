## Top Stories

### [Iroh 1.0](https://news.ycombinator.com/item?id=48542480)
Number0 released version 1.0 of Iroh, an open-source peer-to-peer networking stack built on QUIC. It allows applications to dial endpoints by cryptographic keys rather than IP addresses, handling NAT traversal and hole punching automatically. The HN discussion clarifies its positioning against Tailscale: while Tailscale acts as a global device daemon using WireGuard, Iroh is a lightweight Rust library meant to be embedded directly into applications, using standard TLS and QUIC without requiring a centralized coordination server.

### [A backdoor in a LinkedIn job offer](https://news.ycombinator.com/item?id=48546294)
A developer documented a sophisticated phishing attack where a fake recruiter asked them to review a "broken" GitHub repo for a crypto startup. The repo contained a hidden payload that executed arbitrary code simply by running `npm install`. Commenters note this is a known tactic of state-sponsored actors like the Lazarus Group. The thread serves as a stark reminder that running untrusted code during technical interviews is increasingly dangerous, making throwaway VMs a necessity for take-home tasks.

### [Hetzner Price Adjustment](https://news.ycombinator.com/item?id=48540844)
Hetzner announced a significant price increase for new and rescaled cloud servers, with some US instances tripling in cost and EU instances seeing 30-50% hikes. Existing instances are grandfathered in. The community points to the AI boom as the culprit: hyperscalers are buying up the global supply of RAM and storage, leaving budget providers who rely on commodity hardware uniquely vulnerable to the supply chain crunch.
