## Top Stories

### [ESP32s replaced a $120k bowling center system for $1,600](https://news.ycombinator.com/item?id=48968606)
A solo dev bought an abandoned 8-lane bowling alley and rebuilt the scoring and pinsetter controls using $1,600 of ESP32s. The thread pushed back gently. AMF machines from the 1960s always ran on relays. One retired mechanic called it the canonical "third spaces" tech story.

### [Qwen 3.8 launched with 2.4T parameters, weights coming soon](https://news.ycombinator.com/item?id=48966120)
Alibaba unveiled a 2.4T-parameter Qwen 3.8 Max preview, claiming it sits "second only to Fable 5." Thread consensus: Chinese labs are now firmly in the frontier race. Commenters mostly want smaller distillations for local hardware rather than more 2T-parameter monoliths.

### [Claude Code is now running on a Rust-ported Bun](https://news.ycombinator.com/item?id=48966569)
Anthropic shipped Claude Code using a Bun runtime that has been transpiled from Zig to Rust, a roughly 1M-line change merged in under a month. Users report 10% faster Linux startup and notice no functional change. The thread reads as a referendum on whether AI-generated porting counts as engineering.

### [Minecraft Java Edition now uses SDL3](https://news.ycombinator.com/item?id=48967256)
Snapshot 26-3 swaps GLFW for SDL3, bringing native Wayland support and fixing input bugs on Linux. osu! saw similar latency wins on its SDL2-to-SDL3 port. Snapshot ships with known fullscreen crash bugs to be patched later.
