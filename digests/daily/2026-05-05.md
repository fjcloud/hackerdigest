## Top Stories

### [Bun is being ported from Zig to Rust](https://news.ycombinator.com/item?id=48016880)
The Bun JavaScript runtime is undergoing a massive, AI-assisted rewrite from Zig to Rust. This follows Anthropic's recent acquisition of Bun and Zig's strict anti-LLM contribution policy. The HN discussion is heavily split: some view this 500k+ LOC translation as a fascinating case study in LLM-driven refactoring, while others [worry](https://news.ycombinator.com/item?id=48011184) that Anthropic's rapid, AI-generated development pace is destabilizing a core piece of infrastructure.

### [Microsoft Edge stores all passwords in memory in clear text](https://news.ycombinator.com/item?id=48012735)
A security researcher demonstrated that Edge keeps saved passwords unencrypted in memory even when unused. However, the comments quickly pointed out that this is an OS-level architectural limitation rather than an Edge-specific flaw. Chrome and Firefox exhibit similar behavior; any malicious process running in the same user context with memory-read access can dump secrets, highlighting the inherent limits of browser-based password managers.

### [How OpenAI delivers low-latency voice AI at scale](https://news.ycombinator.com/item?id=48013919)
OpenAI detailed their WebRTC and Kubernetes architecture for serving voice interactions globally, heavily utilizing the Go-based Pion library. While the network engineering is solid, commenters noted that the actual bottleneck for natural conversation isn't transport latency, but Voice Activity Detection (VAD). The current implementation frequently interrupts users who simply pause to think, making the low latency actively detrimental to the UX.
