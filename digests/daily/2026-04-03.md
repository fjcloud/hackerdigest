## Top Stories

### [Decisions that eroded trust in Azure – by a former Azure Core engineer](https://news.ycombinator.com/item?id=47616242)
A former Azure Core engineer published a scathing 6-part series detailing the technical debt and leadership failures behind Azure's reliability issues, including nodes requiring 173 management agents and severe host-side vulnerabilities. The HN thread is filled with users corroborating the platform's instability compared to AWS and GCP, though some debate whether the author's approach to escalating the issues was professional.

### [Google releases Gemma 4 open models](https://news.ycombinator.com/item?id=47616361)
Google dropped the Gemma 4 family, featuring a 31B dense model, a 26B MoE, and smaller E2B/E4B variants with native multimodal (voice and image) support. While the larger models are competitive with Qwen 3.5, the HN community is most excited about the smaller models running locally on edge devices, despite some launch-day bugs with tool calling.

### [Cursor 3](https://news.ycombinator.com/item?id=47618084)
Cursor's latest major release shifts the UI heavily toward an "agent-first" workflow, mimicking the chat-centric design of Claude Code and Codex. The HN discussion reveals a growing backlash against "vibe coding" and agent swarms; many developers prefer AI as an augmentation tool where code remains the first-class citizen, with several noting they are switching back to vanilla VS Code to maintain control.
