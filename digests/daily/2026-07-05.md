## Top Stories

### [Leaking YouTube creators' private videos](https://news.ycombinator.com/item?id=48786781)
A researcher disclosed a method to access private YouTube videos. An ex-Google engineer in the thread traced the delayed patch to GRAD: the performance review framework rewards launches, not bug fixes, giving the responsible team little incentive to act. The explanation lit up a sub-thread on whether software practitioners should face formal licensing regimes akin to civil engineering.

### [Google Books (or similar) all book scans – $200k bounty (2025)](https://news.ycombinator.com/item?id=48786838)
Anna's Archive posted a $200k bounty for the complete Google Books scan corpus, framing it as a long-term preservation and AI training effort. The 232-comment thread divided along a familiar line: readers in book-scarce markets called the archive a genuine lifeline, while others pointed out the contradiction in treating large-scale book piracy as a moral virtue while condemning open-source license violations.

### [Potential session/cache leakage between workspace instances or consumer accounts](https://news.ycombinator.com/item?id=48785485)
Claude Code received a security report alleging that conversation context may leak between distinct workspace sessions or between enterprise and consumer accounts. The failure mode described matches the HTTP desync (request smuggling) pattern, where shared upstream connections can misroute API responses between concurrent callers. Anthropic had not issued an official response at time of posting.

### [GPT-5.5 Codex reasoning-token clustering may be leading to degraded performance](https://news.ycombinator.com/item?id=48789428)
A GitHub issue against OpenAI Codex documents that GPT-5.5 appears to cut its chain-of-thought short at 516 tokens roughly 40% of the time, returning wrong answers on prompts it solves correctly when allowed 6,000 to 8,000 thinking tokens. The thread became a running argument for local models: silent, unannounced server-side changes are exactly the failure mode you cannot anticipate from the outside.
