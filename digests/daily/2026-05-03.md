## Top Stories

### [VS Code inserting 'Co-Authored-by Copilot' into commits regardless of usage](https://news.ycombinator.com/item?id=47989883)
Microsoft changed VS Code to insert "Co-Authored-by Copilot" into git commits by default, even if AI features are disabled. The community is outraged by the falsification of commit logs (which are legal and technical records), with commenters noting this could have serious copyright implications since AI-generated code isn't copyrightable. A Microsoft engineer has since apologized and promised a revert.

### [NetHack 5.0.0](https://news.ycombinator.com/item?id=47988776)
The legendary roguelike released version 5.0.0 (skipping v4 to avoid confusion with a popular fork), marking a massive architectural shift. The ancient lex/yacc compilers have been replaced with Lua bindings, making modding much more accessible. The update also includes thousands of bug fixes and controversial balance changes, like nerfing the notoriously easy Valkyrie class.

### [Maryland Is First to Ban A.I.-Driven Price Increases in Grocery Stores](https://news.ycombinator.com/item?id=47992349)
Maryland passed legislation banning "surveillance pricing" in grocery stores—the practice of using AI and consumer data to charge different customers different prices for the same item. The HN discussion debated the economics of price discrimination, with some noting that while it makes sense for digital goods like Netflix in emerging markets, applying it to essential physical goods is deeply unpopular.
