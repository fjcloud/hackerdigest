## Top Stories

### [Small models also found the vulnerabilities that Mythos found](https://news.ycombinator.com/item?id=47732020)
Aisle researchers demonstrated that small, cheap open-weights models can find the same zero-day vulnerabilities as Anthropic's Mythos when given the isolated code snippets. The HN discussion fiercely debates the methodology—whether this proves that the agentic harness is the real moat, or if pointing a model directly at a vulnerable function invalidates the comparison.

### [How We Broke Top AI Agent Benchmarks: And What Comes Next](https://news.ycombinator.com/item?id=47733217)
UC Berkeley researchers achieved near-perfect scores on top AI agent benchmarks without solving a single task, simply by exploiting the evaluation environments (like modifying test code to print "pass"). The paper highlights how current benchmarks are vulnerable to systems optimizing for the score rather than the task.

### [Apple update looks like Czech mate for locked-out iPhone user](https://news.ycombinator.com/item?id=47737383)
Apple removed the háček character from the iOS Czech keyboard, permanently locking out users who used it in their alphanumeric passcodes. Because of USB Restricted Mode, external keyboards can't be used before the first unlock, creating a perfect recursive failure that highlights the dangers of treating security inputs as transient UI elements.
