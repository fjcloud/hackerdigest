## Top Stories

### [Axios compromised on NPM – Malicious versions drop remote access trojan](https://news.ycombinator.com/item?id=47582220)
The massively popular JavaScript HTTP client was compromised when a maintainer's npm credentials were stolen, bypassing CI/CD. Malicious versions (1.14.1 and 1.14.2) inject a fake dependency that runs a postinstall script to drop a remote access trojan. The HN thread is full of mitigation advice, noting that setting `ignore-scripts=true` or configuring a minimum release age in your package manager would have prevented the infection.

### [Claude Code's source code has been leaked via a map file](https://news.ycombinator.com/item?id=47584540)
Anthropic accidentally shipped source maps in their npm package, exposing the proprietary TypeScript source of their Claude Code CLI. The leak reveals unreleased features like an "assistant mode" and an "undercover mode" for employees. More amusingly, HN commenters are roasting the "vibe-coded" architecture—including a 3,167-line `print` function and the use of regex for sentiment analysis—proving that even elite AI coding agents are built on spaghetti code.

### [Artemis II is not safe to fly](https://news.ycombinator.com/item?id=47582043)
Maciej Cegłowski (idlewords) argues that NASA is ignoring severe anomalies with the Orion capsule's heat shield, which lost chunks of ablative material and melted bolts during the uncrewed Artemis I test. He draws direct parallels to the normalization of deviance that caused the Challenger and Columbia disasters. The HN discussion is split between trusting the NASA engineers who signed off on the risk and fearing a repeat of historical bureaucratic groupthink.

### [GitHub backs down, kills Copilot pull-request ads after backlash](https://news.ycombinator.com/item?id=47582984)
Following intense community outrage, GitHub removed a "feature" where Copilot would automatically edit user PR descriptions to insert promotional tips for third-party tools. GitHub claimed it was a "programming logic issue," but HN commenters aren't buying it, pointing out that stealth-editing a developer's PR is a massive breach of trust and a grim preview of AI-driven enshittification.
