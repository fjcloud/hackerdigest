## Top Stories

### [Contractors wipe 96 government databases after firing](https://news.ycombinator.com/item?id=48115438)
Two IT contractors wiped 96 DHS databases immediately after being fired during a remote meeting. It is a spectacular failure of access control, offboarding processes, and background checks, given they had prior felony convictions. The HN thread is baffled that passwords were stored in plaintext, noting this is a textbook case of why immediate access revocation during termination is standard practice.

### [Windows APIs become Linux kernel features to boost gaming](https://news.ycombinator.com/item?id=48087887)
Linux gaming performance is improving because Windows APIs like NTSYNC are being implemented directly in the Linux kernel. Instead of user-space emulation workarounds that cause subtle hitches, native kernel support matches Windows behavior exactly. The community notes the irony of Linux adopting Windows architecture to win, but pragmatism prevails. The real win is not just raw speed, but correctness that fixes edge-case stutters.

### [Deep dive into the $599 MacBook Neo](https://news.ycombinator.com/item?id=48125617)
Apple's new MacBook Neo brings Apple Silicon to a new low price point using an iPhone A18 Pro chip, but suffers from extreme thermal throttling and a controversial 8GB RAM limit. Benchmarks show an 87% reduction in single-core performance under sustained load. Users note that while the 8GB limit is a forcing function against macOS bloat, the thermal throttling is so severe that some are already doing copper shim mods to double performance.
